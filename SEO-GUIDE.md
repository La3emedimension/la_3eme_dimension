## 📋 Bonnes pratiques à suivre

### Pour chaque nouveau produit :
1. **Titre SEO** : Inclure le prix et mots-clés principaux
   - Format : "[Produit] [Caractéristique] - [Prix]€ | La 3ème dimension"
   
2. **Description** : 155-160 caractères max
   - Inclure : caractéristiques principales, prix, bénéfices
   - Mots-clés naturels (pas de bourrage)

3. **Keywords** : 8-12 mots-clés
   - Mix : génériques + longue traîne + locaux
   - Exemple : "vis collimation" + "vis moletées newton M4" + "toulouse"

4. **Alt text images** :
   - Descriptif + mot-clé
   - Exemple : "Miroir parabolique 150mm f/5 testé interféromètre pour télescope Newton"

### Structured Data (Schema.org)
Toujours inclure :
- Product pour les fiches produits
- AggregateRating si avis disponibles
- Offers avec price, availability, priceCurrency

## 🚀 Prochaines optimisations recommandées

### Court terme (1-2 semaines)
1. **Compression images**
   - Convertir en WebP
   - Compresser à 80-85% qualité
   - Réduire 1500.jpg (154KB) et autres grosses images

2. **Breadcrumbs sur toutes les pages**
   - Pages produits : Accueil > Boutique > [Produit]
   - Pages info : Accueil > [Page]

3. **Meta descriptions uniques**
   - Vérifier toutes les pages
   - Chaque page doit avoir sa propre description

### Moyen terme (1 mois)
1. **Blog/Articles**
   - "Comment collimater un télescope Newton"
   - "Choisir son premier télescope : guide complet"
   - "Observation astronomique : débuter avec l'E.R.T"

2. **FAQ page**
   - Questions fréquentes sur produits
   - Structured data FAQPage
   - Bon pour featured snippets Google

3. **Liens internes**
   - Relier produits complémentaires
   - Exemples : miroir → vis collimation, ERT → filtres

### Long terme (3-6 mois)
1. **Vidéos produits**
   - Montage, utilisation
   - Héberger sur YouTube
   - Embed sur pages produits

2. **Backlinks**
   - Partenariats clubs astronomie
   - Articles invités blogs astronomie
   - Listings annuaires qualité

3. **Performance**
   - Lazy loading images
   - CDN pour assets
   - Minification CSS/JS
